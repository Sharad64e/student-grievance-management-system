# Student Grievance Management System

## Aim

To develop a MERN stack Student Grievance Management System that allows students to register, login, submit grievances, view complaints, search grievances, update/delete their own complaints, and logout securely.

## Software Requirements

- Node.js
- React.js
- Express.js
- MongoDB Atlas
- Postman or Thunder Client
- GitHub
- Render

## Frontend Components

- Registration Form
- Login Form
- Dashboard
- Submit Grievance Form
- Grievance List
- Search Bar
- Update/Delete Buttons
- Logout Button

## Backend Features

- Student registration
- Student login
- JWT authentication
- Protected grievance routes
- Only logged-in users can manage their own grievances
- Error handling for invalid login, duplicate email, and unauthorized access

## Database Collections

### users

Stores student name, email, password hash, created date, and updated date.

### grievances

Stores student ID, title, category, description, status, created date, and updated date.
